Hướng Dẫn Sử Dụng Git2HDD

Cách sao lưu dự án từ SSD sang nhiều HDD dự phòng tự động trên Windows

🔍 Git2HDD là gì?

Khi làm việc, bạn thường viết code trên ổ cứng tốc độ cao (SSD). Để đề phòng ổ SSD bị hỏng hoặc mất dữ liệu, bạn muốn copy dự án sang 2-3 ổ cứng thông thường (HDD) khác để dự phòng.

Thay vì mỗi ngày phải tự sao chép thủ công từng thư mục hoặc mở từng file script phức tạp, Git2HDD cung cấp cho bạn một công cụ tự động. Bạn chỉ cần nhấn một nút trên giao diện web, công cụ sẽ tự động nén, đồng bộ và đẩy dữ liệu sang tất cả các ổ cứng HDD cùng một lúc.

🛠️ Bước chuẩn bị (Chỉ cần làm 1 lần duy nhất)

Trước khi sử dụng, máy tính của bạn cần cài đặt 2 phần mềm nền tảng sau:

  1. Cài đặt Node.js:

    Node.js giúp chạy các công cụ viết bằng Javascript trên máy tính.

    • Truy cập trang chủ nodejs.org.
    • Tải phiên bản LTS (khuyên dùng vì tính ổn định).
    • Mở file tải về và nhấn Next liên tục để cài đặt.
  2. Cài đặt Git:

    Git là phần mềm giúp quản lý các phiên bản và lịch sử thay đổi của dự án.

    • Truy cập git-scm.com để tải bản cài đặt cho Windows.
    • Mở file cài đặt, giữ nguyên các lựa chọn mặc định và nhấn Next để hoàn thành.
💡 Lưu ý quan trọng khi cài đặt: Sau khi cài xong cả Node.js và Git, bạn hãy **khởi động lại máy tính** để hệ thống Windows ghi nhận đường dẫn của các phần mềm này.

📦 Cách cài đặt Git2HDD

Để cài đặt Git2HDD, bạn làm theo các bước sau:

1

Mở công cụ Command Prompt (hoặc PowerShell)

Nhấn phím Windows + R trên bàn phím, gõ chữ cmd rồi nhấn Enter. Một cửa sổ màn hình đen sẽ hiện ra.

2

Nhập lệnh cài đặt

Copy dòng lệnh dưới đây, dán vào cửa sổ Command Prompt rồi nhấn Enter:

npm install -g git2hdd

Hệ thống sẽ tải và cài đặt Git2HDD tự động trong khoảng 5-10 giây.

3

Kiểm tra cài đặt thành công

Nhập lệnh sau và nhấn Enter:

git2hdd --version

Nếu màn hình hiển thị số phiên bản (Ví dụ: 1.0.0) thì bạn đã cài đặt thành công!

💻 Quy trình sử dụng bằng Giao diện Web (Khuyên dùng)

Cách dễ dàng nhất để cấu hình và sử dụng là mở giao diện đồ họa trực quan trên trình duyệt:

1

Di chuyển đến thư mục dự án của bạn

Trên cửa sổ Command Prompt, di chuyển đến thư mục chứa code của bạn bằng lệnh cd. Ví dụ:

cd D:\work\my-project
2

Khởi tạo dự án

Nhập lệnh khởi tạo dự án đầu tiên:

git2hdd init

Lệnh này sẽ tự động tạo một file cấu hình git2hdd.config.json bên trong thư mục dự án của bạn.

3

Mở giao diện quản lý trên web

Nhập lệnh khởi chạy server giao diện:

git2hdd gui

Hệ thống sẽ **tự động mở một trang web** trên trình duyệt của bạn (tại địa chỉ http://localhost:3000).

4

Cấu hình các ổ đĩa HDD dự phòng

Trên trang web vừa mở ra:

  • Nhấp vào tab Cấu hình dự án ở thanh menu bên trái.
  • Tại mục "Danh sách HDD backup", nhấn nút + Thêm HDD Backup.
  • Điền đường dẫn đến thư mục backup trên ổ HDD của bạn (Ví dụ: E:\backup\project.git).
  • Bạn có thể thêm nhiều ổ đĩa đích tùy ý (lên tới 10 ổ đĩa).
  • Nhấn Lưu cấu hình ở góc dưới. Giao diện web sẽ tự động chuẩn bị các ổ cứng này giúp bạn.
5

Thực hiện backup hàng ngày

Mỗi khi làm việc xong cuối ngày và muốn backup:

  • Vào tab Chạy Backup / Mirror.
  • Nhập mô tả ngắn gọn cho bản backup (Ví dụ: hoàn thành nút thanh toán). Nếu để trống, hệ thống sẽ tự đặt tên bằng ngày giờ hiện tại.
  • Nhấn nút Bắt đầu Git Backup.
  • Bạn sẽ thấy toàn bộ tiến trình nén và copy dữ liệu chạy trực tiếp trên khung màn hình Terminal màu đen theo thời gian thực.
🎉 Chúc mừng bạn! Từ giờ, mỗi khi cần backup, bạn chỉ cần mở thư mục dự án, chạy lệnh git2hdd gui và nhấn nút **Backup** trên web là xong.

❓ Các lỗi thường gặp và cách xử lý

  • Lỗi: "npm" hoặc "git2hdd" command not found

    Lỗi này do Windows chưa nhận diện được phần mềm vừa cài. Hãy tắt cửa sổ Command Prompt đi, mở lại cửa sổ mới hoặc khởi động lại máy tính.

  • Lỗi: Backup thất bại ở một ổ đĩa

    Hãy kiểm tra xem ổ HDD đó có đang được cắm vào máy tính hay không, hoặc ký hiệu ổ đĩa (Ví dụ: ổ E:\, F:\) có bị thay đổi không. Các ổ đĩa khác vẫn sẽ được backup bình thường.

  • Muốn tắt giao diện web thì làm thế nào?

    Quay lại cửa sổ Command Prompt đen và nhấn tổ hợp phím Ctrl + C, sau đó gõ Y và ấn Enter để tắt server.
